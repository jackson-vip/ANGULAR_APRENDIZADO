Template Variables

Para acessar a variável de template, utilize a referência local (#meuInput) junto com o ngModel.
Valor: {{ meuInput.value }}
No caso de inputs sem o ngModel, é necessário disparar a detecção de mudanças manualmente, por exemplo, com um botão que chama uma função no componente. O ChangeDetectorRef ao ser injetado no construtor do componente, permite chamar o método detectChanges().
Valor: {{ meuInput2.value }}
Através do ngModel com Two-Way Data Binding, o valor do input é atualizado automaticamente. no entanto, a variável de template (#meuInput3) não é necessária nesse caso, pois o valor já está sendo gerenciado pela propriedade do componente (inputValue).
Valor: {{ inputValue }}
