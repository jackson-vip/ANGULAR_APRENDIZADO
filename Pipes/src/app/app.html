Estudando Pipes no Angular

Pipes Nativos do Angular

Pipe Descrição Exemplo
DatePipe Formata datas Data de hoje: {{ hoje | date:'fullDate' }}
UpperCasePipe Transforma texto em maiúsculas {{ texto | uppercase }}
LowerCasePipe Transforma texto em minúsculas {{ texto | lowercase }}
CurrencyPipe Formata valores monetários {{ 1234.56 | currency:'BRL':'symbol' }}
DecimalPipe Formata números decimais {{ 9876543.21098 | number:'1.2-2' }}
PercentPipe Formata porcentagens {{ 0.756 | percent:'1.1-1' }}
JsonPipe Transforma objetos em strings JSON 
{{ jsonPipe | json }}
SlicePipe Extrai uma fatia de um array ou string Array: {{ array | slice:1:4 }}
String: {{ texto | slice:0:5 }}
AsyncPipe Lida com Observables e Promises Observable: {{ observableData | async }}
Promise: {{ promiseData | async }}
TitleCasePipe Transforma texto em formato de título {{ texto | titlecase }}
I18nPluralPipe Exibe conteúdo com base em pluralização {{ pessoas.length | i18nPlural: pluralMapping }}
I18nSelectPipe Exibe conteúdo com base em seleção {{ genero | i18nSelect: generoMapping }}
KeyValuePipe Transforma objetos em arrays de pares chave-valor 
{{ pessoas | keyvalue | json }}
